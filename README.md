# sv.h
[String View](https://en.cppreference.com/w/cpp/string/basic_string_view) for C.

## Quick Start
Just copy [sv.h](./sv.h) into your project and use it like this

```c
#define SV_IMPLEMENTATION
#include "sv.h"
```
